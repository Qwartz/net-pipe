enum STACK_DRIVERS

Driver flags.

This contains all flags used for driver deletion and creation

Parameters
STACK_CALL_FROM_NULL- Throw in cases of the stack trying to return a null component or variable
STACK_CALL_HAS_NO_DRIVER- Throw in cases of trying to load a null or missing driver
STACK_CALL_FORCED_DRIVER- Forces driver to load under all circumstances
STACK_HAS_EXPLICIT_DRIVER- Uses driver under an explicit circumstance.
